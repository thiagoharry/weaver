Examples

This page shows some games written with the Weaver Engine.

Pong

In the first impression, this is just a Pong with very old style graphics. You can control the paddles with the arrow keys or with the keys W and S in 2-players mode.

During the game you can get some items which can improve the game graphics with new skins, blow up the opponent paddle with a bomb , or activate some hallucinogen effect in the screen.
